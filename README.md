# Валентинка — интерактивный сайт

Простая валентинка с вопросом «Хочешь быть моей валентинкой?» и кнопками Да/Нет. Кнопка «Нет» убегает от курсора, на «Да» — конфетти и приятное сообщение.

## Настройка

1. **Фото:** Добавь своё фото в папку как `photo.jpg` (или измени путь в `index.html`).
   Если файла нет, покажется картинка по умолчанию.

2. **Текст:** В `index.html` можно изменить:
   - вопрос — тег с классом `question`
   - приветствие — класс `greeting`
   - сообщение при «Да» — в блоке `successCard`

## Локальный просмотр

Открой `index.html` в браузере.

## Хостинг (бесплатно)
